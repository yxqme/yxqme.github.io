群晖 DSM 使用密钥登录

2018-07-03 Synology NAS

1. 设置里打开家目录服务

graph LR
    A[用户账号]-->B[高级设置]
    B[高级设置]-->C[家目录]

2. 打开 SSH 功能

graph LR
    A[终端机和SNMP]-->B[终端机]
    B[终端机]-->C[启动 SSH 功能]

3. 修复家目录相关文件夹权限

用户家目录权限只能允许用户自己可写

sudo chmod 755 [target]

chmod 700 ~/.ssh

chmod 600 ~/.ssh/authorized_keys

复制公钥到 authorized_keys

ssh-copy-id -i ~/.ssh/id_rsa.pub [target]

4. 设置 SSH 配置

sudo vi /etc/ssh/sshd_config

修改配置

# 去掉注释，打开下面三个配置
#PubkeyAuthentication yes
#AuthorizedKeyFiles .ssh/authorized_keys
#ChallengeResponseAuthentication no

# 可选，禁用密码登录[一定要确定可以SSH登录后在去掉这个，否则以后就登录不了]
PasswordAuthentication no

重启 SSH 服务

sudo synoservicectl --restart sshd

配置快速连接 ssh/config

Host nas
    IdentityFile /Users/yang/.ssh/id_rsa
    Port 22
    User yang
    Hostname nas.xl-inc.cn
