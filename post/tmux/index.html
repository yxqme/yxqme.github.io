tmux

2015-02-02 2 min read

基本操作

# 列出所有可以的快捷键和其运行的 tmux 命令
tmux list-keys

# 列出所有的 tmux 命令及其参数
tmux list-commands

# 列出所有的 session, window, pane, 运行的进程号，等
tmux info

会话

# 新建一个会话
tmux 

# 新建一个带名称会话
tmux new -s myname

# 进入一个会话
tmux a  #  (or at, or attach)

# 进入一个有名称的会话
tmux a -t myname

# 列出所有会话
tmux ls

# 删除一个会话
tmux kill-session -t myname

# 删除所有的会话
tmux ls | grep : | cut -d. -f1 | awk '{print substr($1, 0, length($1)-1)}' | xargs kill

使用 ctrl+b 进入快捷键模式

会话快捷键

d 退出回话
s 列出所有的会话
$ 命名会话

窗口快捷键

c  新增窗口
w  列出所有的窗口
n  下一个窗口
p  前一个窗口
f  查找窗口
,  命名窗口
&  删除窗口

面板快捷键

%  垂直拆分面板
"  水平拆分面板

o  切换面板
q  显示面板编号
x  删除面板
